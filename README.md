# Тестирование формы заказа доставки карты

![Java CI with Gradle](https://github.com/buslaevamariya11-dotcom/card-delivery-test/actions/workflows/gradle.yml/badge.svg)

Этот проект автоматизирует тестирование формы доставки банковских карт, включая проверку динамических дат и валидацию полей.

## Как запустить

1. **Склонируйте репозиторий:**
   `git clone https://github.com/buslaevamariya11-dotcom/card-delivery-test.git`

2. **Запустите приложение (SUT):**
   `java -jar ./artifacts/app-card-delivery.jar`

3. **Запустите все тесты:**
   `./gradlew test`
   
*(Будут выполнены оба сценария: прямой ввод и работа с календарем)*

## Особенности
* **Технологии:** Java 11, Gradle, JUnit 5, Selenide.
* **CI/CD:** Настроен пайплайн в GitHub Actions для автоматического запуска тестов.
* **Сложные сценарии (Задача №2):** 
  * Автоматизирован выбор города через выпадающий список (autocomplete).
  * Реализована логика взаимодействия с виджетом календаря (умный клик по дате и переключение месяца).
